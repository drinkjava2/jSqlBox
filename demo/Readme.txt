All demos need Java8, all demos can use "mvn test" to run unit test.  

* jbooox is a web-app demo, shows how to use jBeanBox + jSqlBox + jWebBox to build a web site
* jsqlbox-in-spring is a web-app demo, shows how to use Spring + jSqlBox to build a web site
* jsqlbox-beetlsql shows how to build a customized SqlTemplateEngine in jSqlBox
* jsqlbox-java8 shows how to use jSqlBox in Java8, as Java8 support Lambda and default method in interface.
* jsqlbox-xa-atomikos demo shows how to use Atomikos with jSqlBox, in the demo used jBeanBox + JSqlBox + Spring +Atomikos.
